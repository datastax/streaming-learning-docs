= Kinesis Source Connector

The Kinesis source connector pulls data from Amazon Kinesis and persists data into an Apache Pulsar topic. 

For more, see https://pulsar.apache.org/docs/io-kinesis-source[Apache Pulsar's Kinesis source documentation^].
